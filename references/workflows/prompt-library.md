# Prompt Library

## A. 受众蒸馏 Prompt

```text
请帮我创建一个用于 PPT 构思的受众画像 Skill。

【受众定义】
- 角色：
- 关注焦点：
- 决策风格：
- 注意力预算：
- 他们最怕看到什么：
- 他们最想看到什么：

【你的角色】
你是这位受众的顾问。当我提出 PPT 构思时，请从这位受众的视角出发，告诉我：
- 哪些信息他们不关心
- 哪些信息必须前置
- 我的表达方式是否符合他们的认知习惯
- 他们可能会追问什么
```

## B. 结构化大纲 Prompt

```text
基于以下信息，帮我生成 PPT 大纲：

【核心信息】：
【叙事结构】：
【页数预算】：
【素材清单】：

输出要求：
- 每页一个观点型标题，不要只是主题词
- 每页 3-5 个要点，每条不超过 20 字
- 标注图表页、图片页、结构图页、纯文字页
- 标注需要 AI 生成视觉素材的页面
- 最后给出如果只剩 3 页应该保留哪些
```

## C. 单页打磨 Prompt

```text
我们来打磨第 X 页：

当前标题：
这页要传达的核心点：
已有素材/数据：

请给我：
1. 3 个标题方案：保守 / 中性 / 大胆
2. 内容要点的最佳组合
3. 与前后页的逻辑衔接
4. 从受众视角看，这页是否有说服力
```

## D. 编程 Agent 生成 Prompt

```text
你是一个 PPT 生成专家。请基于以下标准交付文档和风格规范，生成完整可运行的 PPTX 生成方案。

【PPT内容】
粘贴标准交付文档

【风格规范】
粘贴 style-card 和 slide-type-map 摘要

【输出要求】
1. 输出可运行代码或可编辑 PPTX
2. 每页严格对照内容规范
3. 数据图表页要准确呈现数据
4. 图片页使用提供路径或占位符路径
5. 代码结构清晰，每个函数对应一页
6. 生成后提供修改指南
```

## E. 评审 Prompt

```text
请基于当前受众画像、评审专家卡和 PPT 风格卡，审查以下 PPT 内容：

粘贴逐页标题和要点

输出：
- P0 必改
- P1 建议改
- P2 可优化
- 标题改写建议
- 是否需要新增/删除/合并页面
```
